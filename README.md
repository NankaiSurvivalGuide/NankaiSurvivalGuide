### 介绍

本指南旨在为天津市南开中学的新生提供一份非官方的入学指引，希望能最大程度减轻大家初来乍到的不适应感，助你尽早融入温暖的南开大家庭。_（温馨提示：请重庆南开中学与南开大学的校友移步至相关对应指南。）_

本指南仿照 [上海交通大学生存手册](https://survivesjtu.gitbook.io/survivesjtumanual/li-zhi-pian/huan-ying-lai-dao-shang-hai-jiao-tong-da-xue) 建立。**编者强烈建议各位现在就去拜读一下这份前作。** 我们正努力构建出同样清晰易读的 UI 界面（例如目前使用的 GitBook 平台），不过当前计划是先将内容框架填充完整。_（咕咕咕中……）_

本手册代码开源，您可以在 GitHub 参与维护：[NankaiSurvivalGuide 仓库](https://github.com/NankaiSurvivalGuide/NankaiSurvivalGuide)

* **知识共享协议**：本指南严格遵守 [CC-BY-SA-4.0协议](https://creativecommons.org/licenses/by-sa/4.0/deed.en)。
* **关于回报**：发起者或许无法为所有贡献者提供实质性的物质回报，目前阶段主要依赖大家的“用爱发电”。不过，或许在线下我们可以一起喝杯奶茶、约个饭？
* **关于灵感**：本指南的初步大纲已在 `outline.md` 文件中呈现（_部分内容仍在补充中_）。当你不知道该写些什么时，不妨去大纲里找找灵感。当然，如果你的想法在大纲之外，我们也极为欢迎各类天马行空的投稿！
* **持续更新**：鉴于社会风向与学校管理层等因素常有变动，本指南需要保持不定期更新。非常欢迎有长期运营意愿的同学联系管理员（联系方式见下）。
